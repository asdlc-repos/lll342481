# Overview

A basic API that responds with "Hello, World!" when called, allowing developers to test connectivity and confirm the service is running.

# Personas

- Developer — calls the API to verify it's working and returns the expected response.

# Features

- A developer sends a request to the API endpoint.
- The API responds with "Hello, World!" in the response body.
- The response includes a standard success status code.
- The API is accessible over HTTP.
- Developers can make requests using common tools like browsers or curl.
